Digital Marketing Landing Page for BirdCrew

Features
-  **Responsive Design** 
-  **Contact & Lead Capture Form:** Validated forms that securely send user data to the backend.
-  **Dynamic Components:** Interactive sections for services, testimonials, and pricing plans.
-  **REST API:** Express-based backend to process data & send email notifications.

**Tech Stack**
- React.js (CRA)
- CSS
- Axios (for API requests)
- Quill (for text editing)
**Backend**
- Node.js
- Express.js
- CORS
- JWT Tokens
- Google OAUTH2
- PostgreSQL
- SendGRID
**Hosting**
- Front-end: Netlify
- Back-end: Render 
  >The backend of this application is hosted on **Render's free tier**.
 If the app has been inactive, the server automatically goes to sleep.
 As a result, **the first API request (such as logging in or submitting a form) may take around 50–60 seconds** to wake up the server.  Thank you for your patience!
